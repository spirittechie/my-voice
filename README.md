# My Voice
Lightweight fatigue-friendly voice appliance for Linux.

## Quick Start
1. sudo dnf install espeak-ng wl-clipboard xdotool
2. pip install --user -r requirements.txt
3. ./bin/build.sh
4. ./bin/myvoice

Hotkeys supported via listener (Super+W for record/paste, Super+R for read).

## Usage
Launch GUI, use buttons or hotkeys for STT/TTS and auto-paste.
